# AutoPalp-server

Node.js server for connecting Meta Quest and AutoPalp Algometer.

## Quick Start

Make sure the AutoPalp algometer is connected.

```
git clone git@github.com:makelab-kaist/AutoPalp2-server.git
cd AutoPalp2-server
pnpm i
pnpm run start
```
